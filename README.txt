Instruções de uso

o Aplicativo foi feito com React Native, então precisará ter o android studio instalado

Utilizando o emulador do Android Studio ou Seu proprio celular com Depuração ativada, abra
 o Prompt de Comando, entre na pasta raiz do aplicativo, e digite react-native run-android.

Aguarde a instalação.

Em seguida abra a aplicação e aperte o botão Acessar.

Você será redirecionado para a tela de login, Não precisa realizar nenhum cadastro.

Logando no sitema você pode verificar que tem Três opções 1° Adicionar Usuario, 2° Visualizar Usuarios, 3° Repositorios

Todos usuarios adicionados são listados na opção visualizar, que pode também  adicionar e remover.

Na opção 3 você pode usar o link do seu git-hub para listar seus projetos "Ex Linykeer/Test-Incca", podendo também atualiza-lo.
